# Grove - Moisture Sensor

これは，センサを土などに差し込み，土などが含む水分の量に応じて，センサの出力電圧が変化するタイプのセンサである．

- Grove - Moisture Sensor 製品ページ :  https://wiki.seeedstudio.com/Grove-Moisture_Sensor/

### センサID
この数字を変更することで，多数のセンサが交じる環境で，このセンサのデータを特定することが可能．
